Loopback Social

Loopback Social은 커뮤니티 간의 연결을 위한 플랫폼입니다.

이 웹사이트를 통해 커뮤니티 운영자는 각 커뮤니티 상단에 부착할 수 있는 검은색 띠 배너를 받을 수 있습니다. 이 배너에는 캠페인에 동참하는 다른 커뮤니티의 이름이 함께 표시됩니다.

이를 통해 커뮤니티 간의 연결을 상징하고, 함께하는 캠페인으로서 운영하고자 합니다.

설치 방법

웹 사이트나 포럼 상단에 배너를 추가하려면 다음 한 줄의 코드를 HTML 파일이나 스킨 본문에 </body> 태그 (닫는 body 태그) 바로 앞에 추가하면 됩니다.

<script src="https://loopback.social/banner.js" defer></script>

참여 방법

커뮤니티를 배너에 같이 표시되게 추가하려면, 배너 설치 후 GitHub 저장소에 이슈를 제출해주세요.

작동 원리

banner.js 스크립트는 웹사이트에 포함될 때 페이지 상단에 검은색 띠 배너를 동적으로 생성하고 삽입합니다. 이 배너에는 loopback.social이라는 프로젝트 제목과 참여 커뮤니티 목록이 포함된 드롭다운 메뉴가 표시됩니다. 커뮤니티 목록은 banner.js 파일 내에서 관리됩니다.

이 프로젝트는 MIT 라이선스를 따릅니다.
